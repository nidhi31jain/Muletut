Using Map over multiple elements

  • Change the input to the below code
    					
<students>
	<student>
		<firstname>Jane</firstname>
		<lastname>Doe</lastname>
		<dob>13th July 2006</dob>
	</student>
	<student>
		<firstname>Emma</firstname>
		<lastname>Watkins</lastname>
		<dob>1st November 2003</dob>
	</student>
</students>
  • In the editor, change dataweave to the below code
    					
%dw 1.0
%output application/json
---
payload.students map {
	'student$$': {
		name: $.firstname ++ " " ++ $.lastname,
		date-of-birth: $.dob
	}
}
